About
My goal with this site is to use as few third party tools as possible. I do use product libraries like Telerik and obout regularly for professional projects. However, I feel that doing as much as possible without them will provide a better learning experience. Then if I do use third party tools for work related projects, I will be able to use them more effectively because I have a better base understanding of how the code works.
Technology Used
This site is coded using ASP.NET MVC, C# and jQuery. The database is MySQL. Since this is a personal site I am using HTML5 and am not trying to support older browser versions. The top menu data is read from an XML file. I just felt like trying XML for the menu and thought it might be easier to manage the menu for a small site via XML rather than creating database structures.
