About
This is my personal site for learning new things. The best way to learn is to have a problem to solve, so I created this site. I wanted a place to try things that I could create from the ground up. I felt that this would provide a better learning experience. That way when I get into other projects or use third party tools I will have a better understanding of how the code works.
Technology Used
This site is using ASP.NET Web Api, C# and Angular 5. The database is MySQL. Since this is a personal site I am not trying to support older browser versions. The top menu data is read from an XML file just because I felt like it at the time. Most of the data is stored in the database.
